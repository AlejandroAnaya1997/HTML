LISTAS

Para crear listas utilizamos dos etiquetas.

< ul > y < /ul > para listas no ordenadas (señaladas por bolitas o cuadrados).

< ol > y < /ol > para listas ordenadas (señaladas por números o letras).

En ambos casos, cada uno de los términos o elementos (ordenados o no) que van a componer la lista se deben encerrar entre las etiquetas < li > y < /li >.

LISTA NO ORDENADA

  • Yeison
  • Pedro
  • Andrea
  • Lina

LISTA ORDENADA

  1. Andrea
  2. Lina
  3. Pedro
  4. Yeison

TIPOS DE VIÑETA

< ul type="circle" > (visualiza círculos sin relleno como viñetas).

  • Andrea
  • Lina
  • Pedro
  • Yeison

< ul type="square" > (visualiza cuadros con relleno como viñetas).

  • Andrea
  • Lina
  • Pedro
  • Yeison
    Tipos de Viñetas
  1. ' circle '
  2. ' square '
  3. ' disk '

TIPOS DE LISTAS ORDENADAS

A: para letras mayúsculas.

a: para letras minúsculas.

I: para números romanos en mayúscula.

i: para números romanos en minúscula.

< ol type="A" > (se crea una lista ordenada alfabeticamente por letras mayúsculas).

  1. Andrea
  2. Lina
  3. Pedro
  4. Yeison

< ol type="a" > (se crea una lista ordenada alfabeticamente por letras minúsculas).

  1. andrea
  2. lina
  3. pedro
  4. yeison

Además, podemos indicar por qué número o letra tiene que empezar a "contar", añadiendo el atributo ' start: ' en este caso desde el número 4 o la letra D

  3. Pedro
  4. Yeison
